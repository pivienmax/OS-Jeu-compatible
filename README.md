Conception d'un Système d'exploitation depuis le langage Jack (java-like), avec implémentation d'un traducteur Jack -> assembleur, 
dans le cadre d'Architecture des Ordinateurs 1 et Systèmes Mono Tache 1.

Réalisation d'un jeu fonctionnant sur l'OS par la suite.

Focus global sur la gestion de la pile, l'arithmétique binaire et l'assembleur.
